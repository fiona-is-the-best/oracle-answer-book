## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to consult the oracle.

## How to Use

1. Think of your question silently in your mind
2. Click the "Ask Me A Question" button
3. Wait for the oracle to consult the ancient wisdom (2 seconds)
4. Receive your cryptic answer
5. Interpret the wisdom as it applies to your situation
6. Ask again for new insights

- **Next.js 15** - React framework with App Router

## Customization

You can add more oracle answers by editing `app/api/answer/route.ts` and adding to the `oracleAnswers` array.
